## Installation

Install my-project with npm

```bash
1. Clone the repository.

2. Install the requirements:
    npm install

3. Create .env File:
    PORT=
    DB_USERNAME=
    DB_PASSWORD=
    JWT_SECRET=

3. Start the project:
    npm start


Note: Please create the .env file in the root directory.
```

## Tech Stack

**Server:** NodeJS, ExpressJS, MongoDB
